# File Routines Library

## Description

The **File Checkers** are classes for checking various file types, including ZIP, JPEG, PNG, and video files. This library allows you to easily integrate file validation functionality into your Java applications.

## Installation

### Maven

Add the following dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>org.alexvagin</groupId>
    <artifactId>filedom</artifactId>
</dependency>
